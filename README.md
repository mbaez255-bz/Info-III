## README para GitHub 

### Descripción
Este repositorio reúne implementaciones educativas en Java de estructuras de datos y algoritmos de ordenamiento, desarrolladas para el curso de la Universidad IUA. 
Incluye ejemplos ejecutables y una interfaz de línea de comandos (demos) para probar comportamiento y ver trazas.

### Estructura del repositorio
El proyecto sigue una organización coherente entre trabajos prácticos (TP1 a TP7) y cada uno mantiene la siguiente estructura de carpetas:

TPx/src/

    --util/        # Funciones auxiliares y clases de soporte (lógica, estructuras, utilidades
    
    --main/        # Clases con método main() para ejecutar demos o pruebas desde consola
    
README.md         # Descripción breve del TP (opcional)

### Temas principales por TP
TP1 — Tareas y estructuras básicas

      Operaciones elementales en Java (condicionales, bucles, arrays).
      
      Primeros ejercicios con estructuras simples y funciones.
      
      Introducción al uso de paquetes y organización modular del código.

TP2 — Funciones recursivas y utilidades

      Implementación de funciones recursivas (factorial, Fibonacci, potencias, etc.)
      
      Operaciones sobre cadenas y conversiones numéricas.
      
      Cálculo de MCD, conversión de bases y otros algoritmos matemáticos.

TP3 — Pilas y colas

      Implementación de Pilas (Stack) y Colas (Queue) desde cero.
      
      Versiones basadas en arrays y listas enlazadas.
      
      Menús interactivos en consola para probar cada estructura.

TP4 — Listas enlazadas

      Lista simplemente enlazada, doblemente enlazada y circular.
      
      Inserción, eliminación, búsqueda y recorrido.
      
      Demos con trazas de operaciones.

TP5 — Árbol AVL

      Implementación completa de un Árbol AVL.
      
      Inserción, eliminación, rotaciones y balanceo automático.
      
      Verificación de alturas y recorridos (inorden, preorden, postorden).

TP6 — Árbol Rojo-Negro

      Estructura de árbol balanceado con reglas de color.
      
      Operaciones de inserción y rotación.
      
      Ejemplos de prueba y análisis de comportamiento.

TP7 — Montículos (Heaps)

      Implementaciones de MinHeap y MaxHeap.
      
      Métodos de inserción, eliminación y heapify.
      
      Demos en consola para observar trazas paso a paso.
