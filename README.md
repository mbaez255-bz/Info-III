## README para GitHub 

### Descripción
Este repositorio reúne implementaciones educativas en Java de estructuras de datos y algoritmos de ordenamiento, desarrolladas para el curso de la Universidad IUA. 
Incluye ejemplos ejecutables y una interfaz de línea de comandos (demos) para probar comportamiento y ver trazas.

### Estructura del repositorio (resumen)


### Temas principales por TP
- TP1: tareas y estructuras básicas.
- TP2: funciones recursivas, operaciones sobre cadenas y utilidades (MCD, conversiones, etc.).
- TP3: colas y pilas (implementaciones y menús).
- TP4: listas enlazadas.
- TP5: árbol AVL (inserción, eliminación, rotaciones y verificación).
- TP6: árbol rojo-negro (implementación y pruebas).
- TP7: montículos (heaps): `MinHeap`, `MaxHeap` y demos interactivos.
